# 操作文档

本文档提供操作的定义。

## 原子操作

在 Hound Whiteboard 里有六种原子操作。

### 添加对象

向某页中添加某对象。要记录页 id 和对象 id。并全量存储对象内容。

### 删除对象

向某页中删除某对象。要记录页 id 和对象 id。并将对象后移入 history/trash/。

### 修改对象版本

修改某对象的某属性。要记录对象 id 和属性的变化。并将对象内容更新。

### 添加页面

添加某页面。要记录页 id。并全量存储页面内容。

### 删除页面

删除某页面。要记录页 id。并将页面及其上所有对象移入 history/trash/。

### 修改页面版本

修改某页面的某属性。要记录页面 id 和属性的变化。并将页面内容更新。

## 分子操作

将原子操作各种组合就变成了分子操作。
