# 显示器组件文档

本文档提供白板中的重要组件——显示器组件（Monitor）的概述。

### 显示器组件

显示器组件是一个特殊的设备，它可以下辖其它的设备。显示器组件没有输入传感器，而它会输出一个 `ReactElement`。这个 `ReactElement` 内有一个 `canvas`，还有它下辖的设备的其它 `React` 控件。

注：现阶段仍输出 `HTMLCanvasElement`

### 设备树

设备树是由各不同状态的设备挂载到显示器组件上后形成的树形结构，它可以将状态显化并封存在树上的节点中。在 Core-UI Interface 中，事件的接收者是 Board，但可以通过节点定位将信息分发到各节点中。

设备树的例子详见设备文档中[设备的状态压缩](../../devices/docs/device-document.md#设备的状态压缩)部分。
