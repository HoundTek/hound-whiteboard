# 取景框模板（Frame-template）设计文档

> [!WARNING]
>
> 当前仓库中没有与本文完整对应的模板运行时代码。本文应视为**概念设计文档**，描述 Frame template 的目标形态，而不是当前已经落地的功能说明。

## 概述

取景框模板是一组动态生成的、具有特定空间结构的取景框集合。模板对外部系统（如导引链）呈现为单个节点，内部拥有独立的镜头移动行为。模板可以将外部文档页面转化为白板对象并铺展到其内部的取景框中，从而在无限画布上提供结构化的分页编辑体验。

## 术语定义

| 术语               | 定义                                                           |
| ------------------ | -------------------------------------------------------------- |
| **取景框模板**     | 动态生成取景框集合的特殊对象，包含预设的空间结构和镜头移动方式 |
| **模板实例**       | 应用模板后在画布上创建的具体实体                               |
| **模板内部取景框** | 由模板生成并完全附属于模板的取景框，属性受限                   |
| **疆域**           | 模板实例所占据的画布范围，可随镜头移动行为动态扩展             |
| **临时扩展**       | 镜头移出当前疆域时自动产生的临时区域，回退即消失               |
| **真实扩展**       | 编辑或人为固定后持久化的疆域扩展                               |
| **分页编辑**       | 嵌入文档后用户以"页"为逻辑在对应取景框内操作的自然编辑模式     |

## 核心定位

### 3.1 本质

取景框模板是一个**特殊的取景框生成器与容器**。它定义了一组取景框的空间结构和镜头移动规则，对外呈现为单一节点，对内管理附属取景框的完整生命周期。

### 3.2 具体类型

| 类型         | 空间结构     | 取景框特征         |
| ------------ | ------------ | ------------------ |
| **胶卷模板** | 单行水平排列 | 等大矩形，线性延伸 |
| **表格模板** | 二维网格排列 | 等大矩形，行列分布 |
| **树形模板** | 暂不实现     | -                  |

### 3.3 核心特性

- **动态生成**：模板内部的取景框由模板按规则生成，非手动逐一创建
- **结构预设**：空间排列方式由模板类型决定
- **移动方式内置**：模板拥有专属的镜头移动控件，不同模板类型提供不同的移动方式
- **对外黑盒**：外部系统（如导引链）不感知模板内部结构，模板仅作为一个节点

## 模板内部取景框

### 4.1 附属关系

模板内部的取景框**完全附属于模板**，与普通取景框有本质区别：

- 生命周期由模板管理（删除模板时，内部取景框一并删除）
- 不可脱离模板独立存在于画布上
- 不可被外部导引链直接引用为节点

### 4.2 属性限制

| 属性     | 限制                                        |
| -------- | ------------------------------------------- |
| **位置** | 由模板的空间结构决定，不可自由移动          |
| **尺寸** | 由模板规则统一设定，不可单独调整            |
| **旋转** | 不可单独旋转                                |
| **锁定** | 不可单独锁定或解锁                          |
| **名称** | 由模板规则自动生成（如"第1页"、"A1单元格"） |
| **删除** | 不可单独删除                                |

### 4.3 编辑权限

- 模板内部取景框的编辑权限**暂不开放**
- 用户不可单独调整内部取景框的任何属性
- 未来可能开放受限的编辑能力

## 动态扩展

### 5.1 扩展机制

模板的疆域不是固定的，可随镜头移动行为动态扩展。扩展分为两种类型：

### 5.2 临时扩展

- **触发条件**：镜头移动超出当前模板疆域边界
- **行为**：自动生成临时的取景框以填充镜头可视区域
- **生命周期**：临时存在，当镜头回退到已有疆域时，临时扩展自动消失
- **持久性**：不产生任何持久化影响，不修改模板数据

### 5.3 真实扩展

- **触发条件**：用户在临时扩展区域内执行编辑操作，或显式执行固定操作
- **行为**：临时扩展的疆域转变为真实的、持久化的模板疆域
- **持久性**：永久存在，修改模板数据

### 5.4 扩展方向

| 模板类型     | 扩展方向                |
| ------------ | ----------------------- |
| **胶卷模板** | 沿水平方向扩展（左/右） |
| **表格模板** | 沿二维方向扩展（行/列） |

## 嵌入文档

### 6.1 功能描述

模板支持将外部文档（如 PDF）的页面转化为通用白板对象，按规则有序地铺展到模板内部的取景框中。各页面与取景框一一对齐，形成结构化的文档视图。

### 6.2 行为

- 文档页面被提取为白板对象，放置于对应的取景框内
- 页面按文档原有顺序填充模板的取景框序列
- 若文档页数超出模板当前取景框数量，触发疆域真实扩展以容纳全部页面

### 6.3 分页编辑

分页编辑不是独立功能，而是嵌入文档后自然产生的编辑模式。用户以"页"的逻辑在对应取景框内操作，这种体验使用户能够以分页思维使用无限画布，降低认知负担。

## 镜头行为

### 7.1 模板内移动

进入模板实例后，镜头由模板专属的移动控件接管。不同模板类型提供不同的镜头移动方式：

| 模板类型     | 移动方式                   |
| ------------ | -------------------------- |
| **胶卷模板** | 水平滑动，逐帧浏览         |
| **表格模板** | 二维平移，在网格中自由移动 |

### 7.2 控制权

- 模板只需告知镜头设备"当前聚焦的取景框"，不直接操控镜头
- 镜头设备根据模板提供的目标执行平滑移动和变焦
- 镜头控制权流转的细节由镜头设备体系定义（本文不展开）

## 子导引链

### 8.1 定义

模板实例内部可创建子导引链，用于在模板内部取景框之间定义演示路径。

### 8.2 规则

- **归属**：子导引链属于模板实例，需手动定义
- **节点范围**：子导引链的节点只能是**该模板实例内部的取景框**
- **外部引用**：禁止引用模板之外的任何取景框
- **嵌套限制**：不允许嵌套（子导引链内不能再创建子导引链）
- **与父链关系**：父级导引链将模板实例视为黑盒节点，不感知子导引链的存在

## 与其他模块的关系

| 模块              | 关系                                                           |
| ----------------- | -------------------------------------------------------------- |
| **取景框**        | 模板生成附属性的取景框集合，与普通取景框在属性和生命周期上不同 |
| **导引链**        | 模板实例可作为导引链的单个节点；模板内部可创建子导引链         |
| **镜头/设备体系** | 模板提供"当前聚焦取景框"的信息，镜头设备执行移动和变焦         |
| **元素系统**      | 模板内部取景框与元素的关系与普通取景框一致（包含判断，无绑定） |
| **文档导入**      | 模板是文档嵌入的目标载体，将文档页面铺展到内部取景框           |
