# 设备文档

## 设备的定义

在现实生活中，设备（device）通常指电脑的硬件中的输入设备和输出设备、如键盘、鼠标、显示器等。在 Hound Whiteboard 中，设备不再局限于真实的、物理的，而是涵盖了一切可以抽象成一个拥有一些输入和输出传感器的集合。比如刚刚提到的键盘，它可以被抽象为 104 个按键（或者 98 个等）的集合。一支 S-Pen 可以被抽象为一个位置传感器、一个压力传感器、一个角度传感器和一个按键的集合，尽管位置、压力和角度都是由屏幕获取的，但它们都属于这支 S-Pen。而不真实存在的设备，如用于翻页的滑轮，它可以被抽象为一个位置传感器和一个输出控件的集合，因为它会向屏幕上显示自己当前的页码，同时用户也可以滑动它来翻页。
