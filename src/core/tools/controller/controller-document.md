# 控制杆文档

本文档提供控制杆的概述。

## 何为控制杆

控制杆是绑定在某个对象上的，可以拖拽的页面元素。拖拽控制杆往往会改变它绑定的对象的某属性，当然对象的属性限制也会决定这个控制杆能否被拖动。

在 HoundWhiteboard 中，控制杆会作为一个 DOM 对象显示在屏幕上以便用户拖动。

## 控制杆的类型

控制杆是多种多样的，有的可以改变大小，有的可以改变方向，更有甚者可以操控维度！为了区分这些控制杆，我们将控制杆分了类。不同类型的控制杆有着不同的作用和外观。

### 顶点控制杆

顶点控制杆用于对象的顶点调整操作，视觉上通常附加在多边形的顶点处。当然这并不意味着它是多边形对象专属的，它可以被用于调整任何有多边形出现的对象的顶点！
